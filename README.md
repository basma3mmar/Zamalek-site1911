# Zamalek SC Fan Website ⚽️

This is a simple fan website dedicated to Zamalek Sporting Club, one of the biggest football clubs in Egypt and Africa.

## 🌟 Features
- Home section with introduction about Zamalek SC
- Video popup showing fans atmosphere 🎥
- Photo gallery of Zamalek players ⚽️
- Clean and responsive design using HTML & CSS
- Simple navigation with smooth links

## 📸 Gallery
The website includes a collection of Zamalek players with short descriptions about each player.

## 🎥 Video Section
A popup video that shows the passion and energy of Zamalek fans.

## 🌍 About
This project was created as a simple front-end practice using HTML, CSS, and JavaScript.

## ⚽ Note
This is a fan-made project and is not officially affiliated with Zamalek SC.
